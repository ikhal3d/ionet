⚡ Awaiting deployment

BGP Looking Glass

show ip bgp <prefix>, peer summary, AS-path views, plus ping & traceroute from our routers. Powered by hyperglass at lg.ionet.com.au.

⚡ Awaiting deployment

Traceroute & Ping (Live)

Real-time hop-by-hop ICMP path tracing from our PoP, with rolling RTT & loss per hop. Output rendered as a hop table plus an inline sparkline per hop.

⚡ Awaiting deployment

Port Reachability Check

Single host, single port from a small allowlist (22 / 25 / 53 / 80 / 443 / …). Cloudflare Turnstile captcha required, rate-limited per IP, RFC1918/loopback rejected. Deliberately narrow — this is not a scanner.

⚡ Awaiting deployment

DNS / DNSSEC Lookup

A, AAAA, CNAME, MX, TXT, NS, SOA, CAA, SRV, PTR, DS, DNSKEY queries via dnspython. Optional DNSSEC chain validation. Configurable resolver (default 1.1.1.1, 8.8.8.8, 9.9.9.9).

⚡ Awaiting deployment

WHOIS / RDAP

Modern RDAP-first lookups for domains and IPs (proper structured JSON), with python-whois fallback for legacy gTLDs & ccTLDs that haven't migrated.

⚡ Awaiting deployment

ASN / Prefix Lookup

AS holder, country, announced prefixes; or IP/prefix → origin ASN. Backed by RIPEstat (free, fair-use).

⚡ Awaiting deployment

TLS / SSL Inspector

Live handshake against host:port; reports the certificate chain, expiry countdown, key type / size, signature algorithm, and supported ciphers. Reports what the server actually presents — doesn't enforce trust.

⚡ Awaiting deployment

HTTP Security Headers

Grades CSP, HSTS, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy. Returns A+ to F with a per-check score and rationale, like Mozilla Observatory.

⚡ Awaiting deployment

AU Outage Status

Aggregated NBN Co · Telstra · Optus · TPG · Aussie Outages feeds, refreshed every 5 minutes. Surfaces incidents on the AU command-centre map as red dots with tooltip detail.
