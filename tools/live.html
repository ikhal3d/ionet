🔒 Browser tool

What is my IP?

Discover your apparent public IP, reverse DNS, city/country, timezone, and the ASN announcing your network — all from your current connection. Calls ipapi.co with ipify as a fallback. Runs in your browser; nothing is sent through the ionet backend.

Port Reachability Check

Probe up to 10 TCP ports on a single host in parallel from the ionet network. Any port 1-65535. Rejects RFC1918 / loopback / link-local destinations. Not a scanner — bounded surface, captcha-gated, rate-limited at the Cloudflare edge.

Quick presets:

DNS / DNSSEC Lookup

Authoritative DNS queries via dnspython, optional DNSSEC chain validation, configurable resolver.

WHOIS / RDAP

RDAP-first lookups (modern, structured) with python-whois fallback for legacy TLDs.

ASN / Prefix Lookup

AS holder + announced prefixes for an ASN, or origin ASN(s) for an IP / prefix. Backed by RIPEstat.

TLS / SSL Inspector

Live handshake — certificate chain, expiry countdown, key type / size, signature algorithm, and negotiated cipher. Reports what the server actually presents.

HTTP Security Headers

Grades HSTS, CSP, X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy — the Mozilla-Observatory checklist. Returns A+ to F with a per-check breakdown and a fancy gauge.

AU Outage Status

Aggregated NBN Co · Telstra · Optus · TPG · Aussie Outages feeds, refreshed every 5 minutes by the outage-aggregator container.

Auto-loads when this section comes into view.

BGP Route Inspector

Origin AS, prefix announcements, RPKI validity, and sample BGP routes from RIPE RIS collectors worldwide. AS path positions are numbered origin-first (1 = the AS that announced the prefix; higher numbers = each AS that received it next).

Honest framing: until ionet has its own BGP gear, this proxies RIPE RIS instead of running show ip bgp on a router we own. RIPE has no AU collector — closest is Singapore (RRC23) or Tokyo (RRC06). Same data quality, public source.

Traceroute (Live)

Layer-3 hop-by-hop path from the ionet PoP to any public host, with per-hop RTT. Runs on the VM via traceroute with CAP_NET_RAW only — no root. Up to 20 hops, 60-second cap.
