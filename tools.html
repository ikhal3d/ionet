IPv4

Subnet / CIDR Calculator

Enter any IPv4 prefix in CIDR notation — get the network address, broadcast, usable host range, mask, wildcard mask, and host count instantly.

IPv4

VLSM Subnet Planner

Allocate variable-length subnets from a parent prefix in optimal address order. Provide a list of required host counts; the planner picks prefixes that fit and assigns them sequentially.

Identity

JWT Decoder

Paste a JSON Web Token to inspect its header and payload. Decoding happens entirely client-side — your token never leaves the browser. Signature verification requires the issuer's key and isn't performed here.

Cryptography

Hash Generator

Compute SHA-1, SHA-256, SHA-384, or SHA-512 over any text. Uses the browser's Web Crypto API — your input never leaves the page.

Cryptography

Hash Reverse Lookup

Cryptographic hashes are one-way functions — there's no mathematical "unhash." What public sites actually do is a rainbow-table lookup: pre-compute hashes of common passwords and look up your input. We do the same thing, locally, against the SecLists top-10,000 common-password list. If your hash matches a weak password we'll find it. If it doesn't, the plaintext is either uncommon or recovery requires hashcat/john with a GPU. The hash you paste never leaves your browser.

Cryptography

HMAC-SHA256 Generator

Compute a keyed-hash message authentication code (HMAC) using SHA-256. Useful for signing webhook payloads, validating signed cookies, or testing an API integration locally.

Encoding

Base64 Encode / Decode

Quick conversion to and from Base64 for credentials, embedded images, API tokens, and anything else that needs round-trip-safe transport.

Encoding

URL Encode / Decode

Percent-encode and decode strings — the right operation for URL query parameters, path components, and form values.

Security

Password Generator

Cryptographically-secure passwords drawn from crypto.getRandomValues. Choose your length and which character classes to include.

Identifiers

UUID v4 Generator

Generate cryptographically-random version-4 UUIDs (RFC 4122). Handy for primary keys, idempotency tokens, and correlation IDs.

IPv4

IP Address Converter

Convert any IPv4 address between dotted-quad, decimal (32-bit integer), hexadecimal, and binary representations.

Diagnostics

What is my IP?

Shows your apparent public IP address, reverse DNS, geolocation, timezone, and the ASN announcing your network. Calls a CORS-friendly public echo service — ipapi.co — and falls back to ipify if that fails. Results render in your browser; ionet doesn't see the request.

Hardware identity

MAC OUI Lookup

Identify the manufacturer behind any MAC address. Backed by the full IEEE OUI registry (39,000+ MA-L assignments). The first lookup pulls the database (~400 KB gzipped, one-time); after that, lookups are instant. Accepts any format — AA:BB:CC:DD:EE:FF, aa-bb-cc-dd-ee-ff, AABBCCDDEEFF, or just the OUI prefix.

Encoding

Text ↔ Hex ↔ Binary

Convert strings between ASCII, hexadecimal, and binary. Useful for embedded debugging, byte-pattern analysis, and CTF-style puzzles.

Server-side tools that run on the ionet Oracle VM. The backend stack is built — Caddy + FastAPI + outage-aggregator containers (see backend/ in the repo) — and wired up to a unified /api/* proxy via Cloudflare. Each tool below goes live the moment the stack is deployed.

Port Reachability Check

Single host, single port from a small allowlist (22 / 25 / 53 / 80 / 443 / …). Cloudflare Turnstile captcha required, rate-limited per IP, RFC1918/loopback rejected. Deliberately narrow — this is not a scanner.

TLS / SSL Inspector

Live handshake against host:port; reports the certificate chain, expiry countdown, key type / size, signature algorithm, and supported ciphers. Reports what the server actually presents — doesn't enforce trust.

HTTP Security Headers

⚡ Awaiting deployment

AU Outage Status

Aggregated NBN Co · Telstra · Optus · TPG · Aussie Outages feeds, refreshed every 5 minutes. Surfaces incidents on the AU command-centre map as red dots with tooltip detail.

⚡ Awaiting deployment

IP Reputation & CVE Recon

Aggregated passive intelligence — no scanning of the target. Combines:

  • AbuseIPDB · abuse confidence score 0–100, recent reports
  • GreyNoise · is this IP a known scanner / benign noise / unknown?
  • VirusTotal · 90+ vendor verdicts on the IP
  • Shodan · open ports, banners, service versions, exposed devices
  • NVD · CVE matching for any service banners returned

Returns a synthesized 0–100 risk score with a per-source breakdown and a list of CVEs ranked by CVSS.
